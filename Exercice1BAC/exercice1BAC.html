Ici, j'ai adapté un exercice de l'épreuve pratique de NSI. Cet exercice était initialement fait en Python. Ici, je l'ai fait en JavaScript

Le programme vous permet de compter le nombre d'occurrences d'un caractère dans un mot.

Pour utiliser le programme :
- Ouvrir la console
- Ecrire : compter(votre caractere, votre mot)

Le résultat donné sera le nombre de fois où le caractère apparait dans le mot.
