Ici, j'ai adapté un exercice de l'épreuve pratique de NSI. Cet exercice était initialement fait en Python. Ici, je l'ai fait en JavaScript

Le programme vous permet de savoir si votre liste est classée dans l'ordre croissant.

Pour utiliser le programme :
- Ouvrir la console
- Ecrire : verifie([votre liste de nombres, séparés par des virgules])

Si le résultat est True, votre liste est classée dans l'ordre croissant, sinon le résultat sera false.
